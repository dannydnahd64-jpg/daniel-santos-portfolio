# Daniel "DNA" Santos — UGC & Creative Portfolio

An interactive, premium, high-retention creative portfolio for Daniel "DNA" Santos (UGC Creator, Creative Director, and Video Editor). This project showcases direct response campaigns, creative specializing, and post-production capabilities in a fluid and visually immersive interface.

## 🧬 Key Features

- **Interactive Phone Campaign Showcase**: 5 responsive mockup mockups simulating viral social channels (`simpletics`, `TinyNature`, `Evolve`, `Cheeter Catcher`, and `Brainrot AI`) with custom looping previews, playback/sound controls, and views analytics.
- **Fluid Video Preview Modal**: Expanded phone modal with back-and-forth switcher buttons (chevrons) for rapid campaign browsing.
- **3D Particle Background**: Dynamic 3D DNA Particle Lattice flowing smoothly behind the content.
- **Symmetrical Design & Grid Systems**: Modern grid arrays, glowing blur accents, and typography crafted to command attention.
- **Contact & Inquiry System**: Direct inbox brief client-transmission form with state transition overlays.
- **Responsive Layouts**: Designed to be responsive across phones (iPhone), tablets (iPad), and large monitors.

## 🛠️ Technology Stack

- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS v4
- **Animation**: Motion (Framer Motion)
- **Icons**: Lucide React
- **Bundler**: Vite

## 🚀 Local Development

Follow these steps to run the portfolio website on your local machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the app:**
   Navigate to the local URL displayed in your terminal (typically `http://localhost:5173`).
